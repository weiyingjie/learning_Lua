## 注释

### 单行

两个减号

```lua
--
```

### 多行

```lua
--[[
中间内容全部为注释
--]]
```

## 标识符

字母数字下划线

<mark>注：最好不要使用下划线加大写字母的标示符，因为Lua的保留字也是这样的。一般约定，以下划线开头连接一串大写字母的名字（比如 _VERSION）被保留用于 Lua 内部全局变量</mark>

## 全局变量

默认变量都是全局的。

在给一个变量赋值后即创建了这个全局变量，没初始化也能访问，值为**nil**，这里的**nil**就类似于NULL，如：

```shell
> print(tmp)
nil
> tmp = 1
> print(tmp)
1
>
```

所以想要删除一个变量就给它赋值为nil即可
